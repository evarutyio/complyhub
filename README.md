# COMPLYHUB - Real Estate Compliance Platform

A beautiful compliance management platform for real estate brokers in UAE.

## Features

- ✅ 3-step onboarding wizard
- ✅ Dashboard with compliance stats
- ✅ Visual calendar
- ✅ Add/manage compliance events
- ✅ Responsive design
